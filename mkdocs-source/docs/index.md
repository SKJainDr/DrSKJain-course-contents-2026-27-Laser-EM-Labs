# Advanced Laser & Optics Learning Lab — User Documentation

Welcome to the companion user guide for the **Advanced Laser & Optics Learning Lab**, an interactive virtual laboratory covering wave optics, geometrical optics, the particle nature of light, fundamental laser-light concepts, a spectrum/photon-energy explorer, and seven industrial laser-processing workshops — **15 interactive experiments in all**, plus a dedicated safety module.

This guide explains the theoretical background of every experiment and workshop in the simulator, walks through the procedure for using each interactive panel, and provides ready-to-use observation tables, graph instructions, and calculation/result templates so that the simulator can be used exactly like a physical laboratory session.

## How to Use This Guide

Each experiment page follows the same structure: **Aim → Theoretical Background → Governing Formulas → Simulator Controls → Procedure → Observation Table → Graph → Calculation → Result → Precautions → Viva-Voce Questions**. Each workshop page follows the analogous structure for an industrial process rather than a measurement experiment.
