# Formula Reference

A quick-reference sheet of the governing formula for every experiment in this lab, grouped by module.

## Electrostatics

| Experiment | Formula |
|---|---|
| Coulomb's Law | $F = q_1q_2/(4\pi\varepsilon_0\varepsilon_r r^2)$ |
| Electric Dipole | $V=p\cos\theta/(4\pi\varepsilon_0 r^2)$, &nbsp; $E_{axial}=2kp/r^3$, &nbsp; $E_{equatorial}=kp/r^3$ |
| Gauss's Law | $\Phi_E = \oint \vec E\cdot d\vec A = Q_{enc}/\varepsilon_0$ |
| Parallel-Plate Capacitor | $C = \varepsilon_r\varepsilon_0 A/d$ |
| Van de Graaff / Conductors | $V_{dome}=Q/(4\pi\varepsilon_0 R)$, &nbsp; $E_{surface}=\sigma/\varepsilon_0$ |

## Magnetostatics

| Experiment | Formula |
|---|---|
| Biot–Savart (finite wire) | $B=(\mu_0 I/4\pi a)(\sin\alpha_1+\sin\alpha_2)$ |
| Circular Coil / Helmholtz | $B(x)=\mu_0NIR^2/[2(R^2+x^2)^{3/2}]$ |
| Ampere's Law (Solenoid) | $B=\mu_0 nI$ |
| Ampere's Law (Toroid) | $B(r)=\mu_0NI/(2\pi r)$ |
| Force Between Parallel Wires | $F/L=\mu_0I_1I_2/(2\pi d)$ |

## Electrodynamics

| Experiment | Formula |
|---|---|
| Faraday's Law | $\varepsilon=-N\,d\Phi_B/dt$ |
| Motional EMF | $\varepsilon=BLv$, &nbsp; $F_{retard}=B^2L^2v/R$ |
| RC Transient | $V_C(t)=\varepsilon_0(1-e^{-t/\tau})$, &nbsp; $\tau=RC$ |
| RL Transient | $I(t)=(\varepsilon_0/R)(1-e^{-t/\tau})$, &nbsp; $\tau=L/R$ |
| LCR Resonance | $\omega_0=1/\sqrt{LC}$, &nbsp; $Q=\omega_0L/R$ |

## EM Waves & Propagation

| Experiment | Formula |
|---|---|
| Maxwell / Plane Waves | $c=1/\sqrt{\mu_0\varepsilon_0}$, &nbsp; $E_0/B_0=c$ |
| Skin Depth | $\delta=\sqrt{2/(\omega\mu\sigma)}$ |
| Brewster's Angle | $n_1\sin\theta_i=n_2\sin\theta_t$, &nbsp; $\theta_B=\arctan(n_2/n_1)$ |

## Electric & Magnetic Materials

| Experiment | Formula |
|---|---|
| Dielectric Polarization | $P=\varepsilon_0(\varepsilon_r-1)E$, &nbsp; $(\varepsilon_r-1)/(\varepsilon_r+2)=N\alpha/3\varepsilon_0$ |
| Magnetic Hysteresis | $H=nI$; hysteresis loss/cycle/volume $=\oint H\,dB$ (loop area) |

## Universal Constants

| Symbol | Quantity | Value |
|---|---|---|
| $\varepsilon_0$ | Permittivity of free space | $8.854\times10^{-12}$ F/m |
| $\mu_0$ | Permeability of free space | $4\pi\times10^{-7}$ T·m/A |
| $c$ | Speed of light in vacuum | $2.998\times10^{8}$ m/s |
| $k = 1/4\pi\varepsilon_0$ | Coulomb's constant | $8.988\times10^{9}$ N·m²/C² |
